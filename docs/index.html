科目コード / Class Code

The class code for the English class, EN(IL1), is LAS.I111-19.

18あるクラスのうち日本語クラスの科目コードは LAS.I111-XX という形式です．XX の個所には表の左端の番号を埋めます．たとえば，12番目の永藤先生のクラスの科目コードは LAS.I111-12 です．英語クラス EN(IL1) の科目コードは LAS.I111-19 です．
